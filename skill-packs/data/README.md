# Data Skill Packs

Cross-stack data store playbooks for document and search backends.

| Skill | Path | When to use |
| --- | --- | --- |
| MongoDB Document Modeling | `mongodb/mongodb-document-modeling/` | Schema design, indexes, multi-tenant queries |
| Elasticsearch Search Patterns | `elasticsearch/elasticsearch-search-patterns/` | Full-text search, reindex, secure APIs |

Pair with `skills/postgres-and-relational-data-modeling` for relational patterns.
