# Fullstack TypeScript Monorepo Preset

## Defaults

- TypeScript strict mode across frontend and backend packages
- Shared contract package for API DTOs and validation schemas
- Workspace-aware build and test caching
- Unified linting and formatting policy
- Per-package ownership and release notes discipline
