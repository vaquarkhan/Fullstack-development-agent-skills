# Fullstack Agent Skills

This repository is an open skill registry for fullstack delivery workflows.

## Start Here

1. Read `skills/using-fullstack-agent-skills/SKILL.md`
2. Select one preset from `presets/`
3. Follow lifecycle commands in `.cursor/commands/`
4. Choose a starter pack from `starter-packs/`

## Core Lifecycle

- `/spec` define scope and contracts
- `/plan` create safe implementation slices
- `/build` implement frontend and backend changes
- `/validate` prove quality and compatibility
- `/review` run frontend, reliability, and security lenses
- `/ship` release with staged rollout and rollback triggers

## Key Directories

- `skills/` workflow skill definitions
- `presets/` stack-specific defaults
- `references/` operational checklists
- `starter-packs/` opinionated bundles by use case
- `agents/` review personas
- `registry/assets.json` machine-readable index
