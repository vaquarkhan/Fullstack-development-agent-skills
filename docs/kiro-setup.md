# Kiro Setup

1. Run `scripts/install.sh --tool kiro` or `scripts/install.ps1 -Tool kiro`.
2. Review steering guidance in `.kiro/steering/README.md`.
3. Load `skills/using-fullstack-agent-skills/SKILL.md` at session start.
4. Select a preset matching your stack under `presets/`.
5. For framework implementation, load matching skills from `skill-packs/`.
6. Apply checklists from `references/` during validate and review phases.

See `docs/fullstack-skills-catalog.md` for the complete 106-skill inventory.
