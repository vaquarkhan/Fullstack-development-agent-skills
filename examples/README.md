# Examples

Runnable-style example packs that map skills to concrete fullstack architectures.

| Example | Stack |
| --- | --- |
| react-nextjs-nestjs-monorepo | Next.js UI + NestJS API + shared contracts |
| saas-multi-tenant-bff | BFF + auth + tenant-aware APIs |
| payments-webhook-platform | Billing service + idempotent webhook processing |
| aws-edge-fullstack-delivery | CloudFront, ALB, API gateway, observability |
| aws-lambda-api-cloudfront-spa | AWS serverless SPA + API pattern |
| azure-apim-functions-static-web-app | Azure APIM + Functions + Static Web Apps |
| gcp-cloud-run-pubsub-nextjs | GCP Cloud Run + Pub/Sub + Next.js |

Each example includes architecture notes, skill routing, and validation commands.
