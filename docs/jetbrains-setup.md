# JetBrains IDE Setup

1. Install the [JetBrains plugin](https://plugins.jetbrains.com/plugin/32167-fullstack-development-agent-skills) or build from `jetbrains-plugin/`.
2. Use **Tools > Fullstack Agent Skills** to install the toolkit (106 skills: 72 core + 34 packs).
3. Pin `skills/using-fullstack-agent-skills/SKILL.md` for AI sessions.
4. For Java work, load `skill-packs/java/spring-boot/` or other JVM packs.
5. Use `references/` checklists during code review and before merge.
6. Match presets under `presets/` to your language and cloud target.

See `jetbrains-plugin/README.md` and `docs/fullstack-skills-catalog.md`.
