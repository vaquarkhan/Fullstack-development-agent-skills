# Azure Fullstack Checklist

- [ ] APIM policies enforce auth, rate limits, and validation
- [ ] Entra ID app roles and claims mapped to authorization model
- [ ] Function retry, DLQ, and idempotency behavior validated
- [ ] Static Web Apps or CDN release and rollback tested
- [ ] Service Bus/Event Grid failure drills completed
- [ ] Application Insights dashboards and alerts updated
