# Gemini Commands Adapter

Use this folder to map lifecycle commands for Gemini-style agent execution.

Recommended mapping:

- `spec` -> `.cursor/commands/spec.md`
- `plan` -> `.cursor/commands/plan.md`
- `build` -> `.cursor/commands/build.md`
- `validate` -> `.cursor/commands/validate.md`
- `review` -> `.cursor/commands/review.md`
- `ship` -> `.cursor/commands/ship.md`
