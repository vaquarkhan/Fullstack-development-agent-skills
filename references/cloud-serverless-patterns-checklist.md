# Cloud Serverless Patterns Checklist

- [ ] Sync call depth and timeout budgets defined
- [ ] Async consumers are idempotent with DLQ handling
- [ ] Secrets and identity use managed mechanisms
- [ ] Environment promotion is automated and auditable
- [ ] Cost guardrails and autoscaling limits configured
- [ ] Multi-region or failover strategy documented where required
