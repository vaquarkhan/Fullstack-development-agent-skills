# Microservice Resilience And Edge Delivery

## Goal

Harden distributed systems with resilience patterns and global delivery controls.

## Core Skills

- microservice-patterns-saga-and-compensation
- microservice-patterns-outbox-and-cdc
- resilience-timeouts-retries-and-circuit-breakers
- api-gateway-and-edge-security
- load-balancer-strategy-and-traffic-distribution
- cdn-caching-and-edge-acceleration-patterns

## Validation Drill

1. Inject dependency timeout and verify circuit behavior.
2. Replay duplicate events and verify idempotent handlers.
3. Run regional failover test for CDN and load balancer paths.
