## Common Issues

- **Skill not loading:** verify `name` in frontmatter matches folder name
- **Install problems:** use `bootstrap.sh` or `bootstrap.ps1` with absolute target path
- **Validation failures:** run `python scripts/validate-skills.py`

## Questions and Enhancements

Open a GitHub issue with:

- target stack (UI, backend, cloud)
- lifecycle phase (`/spec`, `/build`, `/ship`, etc.)
- expected vs actual behavior
