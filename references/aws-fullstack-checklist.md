# AWS Fullstack Checklist

- [ ] API Gateway auth, throttling, and schema validation configured
- [ ] Lambda timeouts, memory, and concurrency limits reviewed
- [ ] Cognito flows tested for login, refresh, logout, and revocation
- [ ] S3/CloudFront cache and purge strategy validated
- [ ] EventBridge/SQS DLQ and replay procedures documented
- [ ] CloudWatch alarms and X-Ray traces cover critical paths
