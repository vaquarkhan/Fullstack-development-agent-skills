# Azure Serverless Fullstack Delivery

Use preset `presets/azure-serverless-fullstack/SKILL.md` and starter `starter-packs/azure-serverless-fullstack-starter.yaml`.

## Architecture

- Static Web Apps for UI
- API Management + Azure Functions for APIs
- Entra ID for identity
- Service Bus/Event Grid for async workflows

## Validation

- Run `references/azure-fullstack-checklist.md`
- Validate APIM policy and token lifecycle tests
