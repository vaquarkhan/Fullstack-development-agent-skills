# JetBrains IDE Setup

1. Clone this repository or install via `bootstrap.ps1` / `bootstrap.sh`.
2. Open `jetbrains-plugin/README.md` for integration notes.
3. Add `AGENTS.md` to your project and pin `skills/using-fullstack-agent-skills/SKILL.md` for AI sessions.
4. Map runbooks: use `references/` checklists during code review and before merge.
5. For backend-heavy work, load presets under `presets/` matching your language and cloud target.
