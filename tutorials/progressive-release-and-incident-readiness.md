# Progressive Release And Incident Readiness

## Goal

Ship safely with staged rollout controls and production incident response discipline.

## Core Skills

- feature-flags-and-progressive-delivery
- fullstack-observability-and-release-engineering
- incident-triage-and-oncall-runbooks

## Release Checklist

- Canary or phased rollout plan approved
- Rollback thresholds and owner assigned
- Dashboards and alerts updated for changed paths
- Post-release verification window scheduled

## Incident Drill

Run one staged failure scenario and validate runbook steps end-to-end.
